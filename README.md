# CSLModsCommon

CSLModsCommon is a shared framework for **Cities: Skylines (CS1)** mod development.

[![Changelog](https://img.shields.io/badge/Changelog-Click%20to%20View-blue)](CHANGELOG.md)

## Localization
CSLModsCommon includes some built-in localization support, to help improve translations, please contribute on Crowdin:
